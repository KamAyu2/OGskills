## 🌟 What is OGskills?

**OGskills** is a free website built for college students — especially complete beginners — who want to figure out where to start in the world of technology.

It gives you a simple way to:
- ✅ See what real-world tech skills exist
- ✅ Mark the ones you've tried or learned
- ✅ Watch free YouTube courses to learn new ones
- ✅ Save your progress to your personal cloud account
- ✅ Track your entire learning journey over time

> *"Where do I even start?"* — That's the question OGskills was built to answer.

---

## 🚀 Features

### 📊 Skill Tracker
Browse through real-world tech skills — HTML, Python, DSA, Git, React, SQL and more. Tap any skill you've tried to mark it green. Watch your progress bars fill up as you learn more. Add your own custom skills with your own proficiency level too!

### 🎬 17+ Free YouTube Courses
Hand-picked YouTube courses across 8 tech domains — completely free. No sign-up needed to watch. Just click a thumbnail and it opens on YouTube instantly.

### ☁️ Cloud Save with Firebase
Create a free account using Google or email. Your tracked skills save to Firestore automatically — always there when you come back, from any device.

### 🌙 Dark & Light Mode
Smooth animated toggle between a clean light theme and a sleek dark theme. Your preference is remembered every time you visit.

### 📱 Fully Responsive
Works perfectly on mobile, tablet, and laptop — no app download needed.

### 🔒 Browse First, Sign In Later
Everyone can browse and explore freely. The sign-in popup only appears when you try to save your progress. Watching courses never requires a login — ever.

## 🔐 How Sign-In Works

| What You're Doing | Login Needed? |
|---|---|
| Browse the website | ❌ No |
| Read all content | ❌ No |
| Watch YouTube courses | ❌ No |
| Tap first 3 skills | ❌ No |
| Tap 4th skill onwards | ✅ Popup appears |
| Add custom skill | ✅ Popup appears |
| Save skills to cloud | ✅ Required |

Sign in options:
- **Google** — one click, instant
- **Email & Password** — create a free account in seconds
